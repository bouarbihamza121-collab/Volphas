║                         VOLPHAS — HOW TO USE                         ║
╚══════════════════════════════════════════════════════════════════════╝

You only need to know ONE thing:

    Double-click  START.bat


That's it. First time it runs, it sets up high-quality voices
(takes 2-3 minutes, needs internet). Every time after, it just
opens the app instantly in your browser.


─────────────────────────────────────────────────────────────────────
  File map of this folder
─────────────────────────────────────────────────────────────────────

  START.bat              ← DOUBLE-CLICK THIS
  volphas-app.html       ← the app itself (opens automatically)
  audio\                 ← voice files
  tools\                 ← scripts the launcher uses
  images\                ← wolf mascot images


─────────────────────────────────────────────────────────────────────
  If something goes wrong
─────────────────────────────────────────────────────────────────────

  "Python is not installed"
    → Install from https://python.org/downloads
    → Tick "Add Python to PATH" during install.
    → Double-click START.bat again.
    → (Or just use START.bat without Python — you'll get robotic
       browser voices instead of Sonia/Ryan, but the app still works.)

  Voices still sound robotic after setup
    → Open the app, press Ctrl+F5 to force-refresh the cache.
    → Make sure you opened the HTML by double-clicking START.bat
      and not by opening some other copy elsewhere on your computer.

  Want to move this whole thing to your Desktop?
    → Go ahead. Just move the ENTIRE "volphas" folder as one unit.
      START.bat will keep working as long as all the files stay
      together in one folder.
